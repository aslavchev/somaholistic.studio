# Security Policy

## Reporting Vulnerabilities

**DO NOT** open public issues for security vulnerabilities.

**Contact:** Repository owner via GitHub private vulnerability report

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.3.x   | ✅        |
| < 1.3   | ❌        |

## Security Features
- CSP Phase 2 (nonce-based)
- HTTPS-only (HSTS)
- No inline scripts
- SRI for external assets
- Zero npm vulnerabilities
